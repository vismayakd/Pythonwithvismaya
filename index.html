Your path

From zero to writing your own programs

Get comfortable with the basics

Variables, data types, and operators — the vocabulary you'll use in every line of Python from here on.

Make your programs think

Conditions and loops let your code make decisions and repeat work — this is where it starts feeling like "real" programming.

Organize your code like a developer

Functions, lambdas, and OOP teach you to structure programs the way professional codebases are actually written.

Handle real-world messiness

Files, errors, and decorators — the practical skills that separate tutorial code from code that actually works.
